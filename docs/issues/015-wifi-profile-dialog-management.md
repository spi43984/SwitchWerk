# Issue #15: WiFi Profile Dialog Management

## Ziel
WLAN-Profile in einem Dialogfeld verwalten, ähnlich wie Geräte.

## Scope
- WLAN anlegen über Dialog
- WLAN bearbeiten über Dialog
- Dialog überlagert die Einstellungen
- Dialog ist scrollbar
- Bestehende Passwortlogik bleibt erhalten

## Nicht im Scope
- Änderung der Passwortspeicherung
- Änderung der Geräteverwaltung

## Akzeptanzkriterien
- [ ] WLAN-Profil anlegen öffnet Dialog
- [ ] WLAN-Profil bearbeiten öffnet Dialog
- [ ] Zurück-Geste schließt Dialog
- [ ] Passwort anzeigen/leeren funktioniert weiterhin
