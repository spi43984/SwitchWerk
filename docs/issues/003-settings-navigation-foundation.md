# 003 Settings Navigation Foundation

## Ziel

Navigation zwischen Dashboard und Einstellungen vorbereiten.

## Scope

- Dashboard bekommt einen Einstieg in Einstellungen
- Einstellungsseite zeigt Platzhalter
- Rueckweg zum Dashboard funktioniert

## Out of Scope

- Speicherung
- Netzwerk
- echte Bearbeitung von Profilen oder Geraeten

## Akzeptanzkriterien

- [ ] Einstieg vorhanden
- [ ] Einstellungsseite vorhanden
- [ ] Rueckweg vorhanden
- [ ] Build erfolgreich
